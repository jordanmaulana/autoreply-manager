## Workflow
- User can manage knowledge. There are title and description.
- That knowledge will be converted to RAG.
- User can connect their Instagram, Threads and Whatsapp.
- Then user can connect those social medias to the knowledge. 1 account, 1 knowledge. 1 knowledge can be used by multiple account.
- Do auto reply to that account by connected knowledge.

## Catch
How to handle unlimited text description. E.g. description is an entire book. Will it halt the database.
